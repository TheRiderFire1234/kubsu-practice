# Генетический алгоритм на C++

Файл: `genetic_algorithm.cpp`

## О программе

Реализован простой генетический алгоритм для минимизации функции
`f(x) = 5x^3 - 4` на множестве значений `x = {1,2,3,4,5}`.

Поддерживаются:
- Бинарное кодирование хромосом
- Стратегии начальной популяции: `одеяло`, `фокусировка`
- Типы селекции: `случайная`, `инбридинг`
- Кроссинговеры: одноточечный, двухточечный, частично соответствующий одноточечный, на основе чисел Фибоначчи
- Операторы мутации/инверсии: точечная, транспозиция
- Отбор: пропорциональный
- Схема эволюции: микроэволюция

## Сборка

Если установлен `g++`, выполните в терминале:

```powershell
cd "c:\Users\ivita\OneDrive\Документы\Practice_2026"
g++ -std=c++17 genetic_algorithm.cpp -O2 -o genetic_algorithm.exe
```

Если используется MSVC, можно собрать с помощью `cl`:

```powershell
cl /EHsc /std:c++17 genetic_algorithm.cpp
```

## Запуск

```powershell
.\genetic_algorithm.exe
```

Программа предложит использовать параметры по умолчанию или задать значения вручную.
